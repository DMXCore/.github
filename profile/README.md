# DMX Core — Professional DMX Lighting Control

[![Website](https://img.shields.io/badge/Website-dmxcore.com-blue?style=flat-square)](https://dmxcore.com)
[![Documentation](https://img.shields.io/badge/Docs-docs.dmxcore.com-blue?style=flat-square)](https://docs.dmxcore.com)

DMX Core builds professional DMX lighting control software and hardware for integrators, system designers, and venue operators. From intimate installs to large‑scale productions, our tools give you flexible, reliable control over complex lighting systems.

---

## Products

### DMX Core 100
A cross‑platform DMX lighting control platform available as **desktop software** (Windows, macOS, Linux) or as a **dedicated wall‑mounted touchscreen controller**.

- Up to **200 DMX universes**
- Protocol support: **ArtNet, sACN, KiNet, OSC, MQTT**
- Timeline editor for building lighting sequences and shows
- Scheduling, automation triggers, and web/API access
- Free demo download available at [dmxcore.com/download](https://dmxcore.com/download.html)

### DMX Core Pico 2
Compact hardware built on the **Raspberry Pi Pico** platform, featuring two robust, isolated DMX512 ports. Perfect for embedded and custom installations.

### DMX Core Pico 2 Re‑Mapper
An enhanced version of the Pico 2 with built‑in software for **DMX channel re‑mapping** — ideal for venues that need flexible patching without reconfiguring fixtures.
